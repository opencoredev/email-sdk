Email SDK exposes separate surfaces for coding assistance and runtime sending.

| Surface | Use it for | Safety boundary |
| --- | --- | --- |
| Coding-agent skill | Teach an agent to integrate, test, and document Email SDK. | It does not send email on its own. |
| `@opencoredev/email-sdk/ai` | Add one approval-gated Vercel AI SDK tool to application code. | Application binds `from`; the model cannot select routing or advanced fields. |
| `@opencoredev/email-sdk-mcp` | Run a local stdio server with validate-reference-approve-send flow. | Sending is absent unless enabled, credentials stay in server env, and elicitation fails closed. |
| `/agent-tools` | Legacy framework-neutral tool shape. | Deprecated for v1; prefer `/ai` or MCP. |
